## System  requirements

**Operating system version:** macOS 10.12 (64 bit) or higher

**RAM:** at least 2 GB

**Browsers:** Safari, Google Chrome, Opera, Yandex.Browser, Mozilla Firefox, and any other macOS compatible browser

**Free disk space:** 120 Mbytes
